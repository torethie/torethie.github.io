Oppgave 1: Publisere nettside med GitHub Pages

Brukerside: brukernavn.github.io. Kort, enkelt og sikkert.

A. Før du starter

  • GitHub‑konto.
  • Valgfritt: GitHub Desktop på Mac, ellers nettleser.

B. Mål på 10 minutter

  • Opprett repo: brukernavn.github.io
  • Legg inn index.html
  • Aktiver Pages (main /root)
  • Sjekk https://brukernavn.github.io/

C. Steg for steg

  1. New repository → navn brukernavn.github.io (f.eks. torethie.github.io), Public, Add a README → Create.
  2. Add file → Create new file → index.html → lim inn enkelt innhold → Commit.
  3. Settings → Pages → Source = Deploy from a branch. Branch = main (/root) → Save.
  4. Åpne https://brukernavn.github.io/.

D. Sjekkliste

  • Repo‑navn matcher eksakt.
  • Public repo.
  • index.html i rot.
  • Pages peker til main /root.
