Oppgave 1: Publisere nettside med GitHub Pages

Målet: Opprett repo brukernavn.github.io, lag index.html, aktiver Pages, sjekk at siden vises.

Steg for steg

  1. Logg inn på github.com → New repository.
  2. Repo-navn: brukernavn.github.io (eks: torethie.github.io), Public, legg til README.
  3. Lag fil index.html → lim inn minimumskode → Commit.
  4. Settings → Pages → Source: main /root → Save.
  5. Vent litt → åpne https://brukernavn.github.io.

Minste index.html

<!DOCTYPE html>
<html><head><meta charset="UTF-8"><title>Hei</title></head>
<body><h1>Velkommen til min første webside!</h1></body></html>

Sjekkliste

  • Repo-navn matcher (brukernavn.github.io)
  • Public repo
  • index.html i rot
  • Pages peker til main /root

Vanlige feil

  • Feil repo-navn → Gi nytt navn
  • Mangler index.html → legg til i rot
  • Feil branch → sett til main /root
  • Forsinkelse → vent og oppdater
