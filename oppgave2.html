Oppgave 2: Lære Bootstrap

Bruk Bootstrap via CDN for å lage knapp, kort og enkel grid‑layout.

Kommentar ved <link>: Laster inn Bootstrap CSS fra CDN slik at du kan bruke klasser som btn, card og container.

Steg-for-steg

  1. Koble til Bootstrap via CDN (CSS + JS).
  2. Lag en h1 med text-center og mt-5.
  3. Knapp: btn btn-primary.
  4. Kort med tittel, tekst og knapp.
  5. Layout med container, row, col (to kolonner ≥ md).
